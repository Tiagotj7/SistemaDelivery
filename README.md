# 📦 Sistema De Delivery

Sistema web de delivery desenvolvido para gerenciamento de pedidos, produtos e clientes. Ideal como projeto de estudo e portfólio, simulando o funcionamento de uma aplicação real de pedidos online.

---

## 🚀 Funcionalidades

### 👤 Cliente

* Visualizar produtos disponíveis
* Adicionar itens ao carrinho
* Realizar pedidos
* Acompanhar status do pedido

### 🛠️ Administrador

* Cadastro, edição e exclusão de produtos (CRUD)
* Gerenciamento de pedidos
* Controle de usuários/clientes

---

## 🧠 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de:

* Praticar conceitos de desenvolvimento web
* Implementar operações CRUD

## ⚙️ Como Executar o Projeto

### 🔧 Pré-requisitos

* Servidor local (XAMPP, WAMP, etc.) ou Node instalado
* Banco de dados (MySQL ou equivalente)

---

### ▶️ Passos

1. Clone o repositório:

```bash
git clone https://github.com/Tiagotj7/SistemaDelivery.git
```

2. Acesse a pasta:

```bash
cd SistemaDelivery
```

3. Configure o banco de dados:

* Crie um banco (ex: `delivery_db`)
* Importe o arquivo `.sql` (se houver)

4. Configure o arquivo de conexão com o banco:

```bash
config/database.php
```

*(ou equivalente no seu projeto)*

5. Inicie o servidor:

* XAMPP → Apache + MySQL
* ou:

```bash
npm start
```

6. Acesse no navegador:

```
http://localhost/SistemaDelivery
```
